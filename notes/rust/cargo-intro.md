# Cargo

## What?

Rust Package Manager

* Packages Dependencies
* Compiles packages
* Make Distributable Packages
* Upload packages to [Crates.io](Crates.io)

## Project File Layout:

* Cargo.lock
* Cargo.toml
* src/
  * Source Code
* benches
  * Benchmarks
* examples
  * Example Code
* tests
  * Integration tests
